# Fibonacci Recursivo

Esta es un ejemplo de la serie fibonacci con recursividad, 
disponible en [GitHub](https://github.com/enevaca/fibonpm).

## Ejemplo de la serie
0 1 1 2 3 5 8 13 21 ....


## Instalación
```
npm install fibonpm
```

## Uso 
```
import fibonpm from 'fibonpm'

fibonpm(5) //3
fibonpm(8) //13
```
